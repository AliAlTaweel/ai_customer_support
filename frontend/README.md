# Customer Support Frontend

This is the Next.js 15 frontend for the AI Customer Support system.

## Getting Started

1.  **Environment Setup**:
    Copy `.env.local.example` to `.env.local` and add your Clerk keys.
    
2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

## Features

- **Clerk Authentication**: Secure signup and login.
- **Order Management**: View and track your orders from the local SQLite database.
- **Modern Support**: AI-powered assistance (coming soon).
